BREVE HISTORIA DE HTML



¿Qué es HTML5?

HTML5 es la versión más reciente de HTML . El término se refiere a dos cosas. Uno es el propio lenguaje HTML actualizado, que tiene nuevos elementos y atributos. El segundo es el conjunto más amplio de tecnologías que funcionan con esta nueva versión de HTML, como un nuevo formato de video, y le permiten crear aplicaciones y sitios web más complejos y potentes.

Para comprender cómo ha evolucionado HTML a lo largo de los años, veamos las diferencias entre HTML y HTML5.



HTML frente a HTML5

HTML es el lenguaje de marcado central de la World Wide Web. Originalmente diseñado para describir semánticamente documentos científicos, desde entonces ha evolucionado para describir mucho más.

La mayoría de las páginas en la web hoy en día se crearon con HTML4. Aunque ha mejorado mucho desde la primera versión de HTML escrita en 1993, HTML4 todavía tiene sus limitaciones. La mayor era si los desarrolladores o diseñadores web querían agregar contenido o características a su sitio que no eran compatibles con HTML. En ese caso, tendrían que usar tecnologías propietarias no estándar, como Adobe Flash, que requería que los usuarios instalaran complementos de navegador. Incluso entonces, algunos usuarios no podrían acceder a ese contenido o función. Los usuarios de iPhones y iPads, por ejemplo, no podrían hacerlo ya que esos dispositivos no son compatibles con Flash.

Señal, HTML5. HTML5 fue diseñado para eliminar la necesidad de esas tecnologías propietarias no estándar. Con esta nueva versión de HTML, puede crear aplicaciones web que funcionan sin conexión, admitir videos y animaciones de alta definición y saber dónde se encuentra geográficamente.

Para comprender cómo HTML5 puede hacer todo eso, veamos las novedades de esta última versión de HTML.



¿Qué hay de nuevo en HTML5?

HTML5 fue diseñado con objetivos principales, que incluyen:

  • Hacer que el código sea más fácil de leer para los usuarios y los lectores de pantalla
  • Reducir la superposición entre HTML, CSS y JavaScript
  • Promover la capacidad de respuesta y la consistencia del diseño en todos los navegadores
  • Compatibilidad con multimedia sin necesidad de Flash u otros complementos


Linea del tiempo de la evolución de HTML

  • 1989: Tim Berners-Lee comenzó a idear el concepto de HTML mientras trabajaba con el CERN. Pronto, Internet fue inventado por él.
  • 1991: Tim mantuvo sus ideas abiertas al público en línea, haciendo todo lo posible para difundir su trabajo.
  • 1992: Joseph Hardin y Dave Thompson se enteraron de las ideas de HTML de Tim. Más tarde, el equipo Mosaic introdujo la etiqueta IMG.
  • 1993: se lanzó la versión 1 de Mosaic Browser para Sun Microsystems's Inc. Esta característica ampliada, incluidas imágenes, formularios y listas.
  • 1994: HTML 2.0 comienza a formarse y se prepara para su lanzamiento. HTML iba a incluir todo lo que tenía 1.0 con nuevas características añadidas
  • 1997 - HTML 3.2 se completa y se lanza al público. Más adelante en el año, se crea HTML 4.0.
  • 2004: la evolución de HTML se reabrió y comenzó a probarse, comenzando con HTML5 y con las primeras propuestas preliminares
  • 2007 - Se crea un grupo de trabajo para trabajar con WHATWG en el desarrollo de HTML5. Apple, Mozilla y Opera permitieron que W3C publicara especificaciones bajo los derechos de autor de W3C.
